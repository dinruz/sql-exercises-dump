# SQL Exercises Dump Repo

![T-SQL](https://img.shields.io/badge/T--SQL-007ACC?style=flat&logo=microsoft&logoColor=white)

Personal collection of SQL queries and scripts. This repository serves as a scratchpad for data exploration and analysis.

## Core Focus

 - **Relational Mapping**: Utilizing `INNER`, `LEFT`, and `FULL JOIN`s to integrate complex data entities.
- **Aggregations & Analysis**: Translating business questions into actionable insights using `GROUP BY`, `HAVING`, and summary functions.
- **Data Hygiene**: Ensuring data quality by identifying and handling NULL values and applying standard formatting.
- **Schema Navigation**: Working with AdventureWorks and film databases to model real-world analytical workflows.

## Technical Details

### **Environment**:
 
  Queries are developed in **SQL Server Management Studio** (SSMS) using **T-SQL**.

 ### **Organization**: 

Each project folder contains:

 * own `scripts/` directory
 * database diagram (ERD) 
 * documentation for context (`data dictionary` and `README.md`).
